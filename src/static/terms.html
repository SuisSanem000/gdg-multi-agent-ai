4. Session Data and Memory Wiping

To preserve session state, this application stores temporary relationship facts in an in-memory instance of ChromaDB. Since the database runs in-memory, session facts are ephemeral and will be wiped if the hosting environment restarts. Users can also manually purge all saved session data at any time by clicking the "Clear Memory" button in the dashboard header.
